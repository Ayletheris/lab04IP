# HouseDash – Lab04 IP

Aplicatie simpla de cazare scrisa in Java, modelata dupa un sistem de tip Airbnb.

## Clase

**Utilizator** – retine datele unui utilizator (nume, email, parola) si permite autentificarea si scrierea de recenzii.

**Proprietate** – reprezinta o locatie de cazare cu pret pe noapte si disponibilitate. Calculeaza costul total pentru un numar de zile.

**Rezervare** – retine datele unei rezervari si proceseaza plata, schimband statusul in confirmat.

**Recenzie** – contine note pentru curatenie si comunicare, si calculeaza media automata.

## Tehnologii

- Java
- Lombok (getteri, setteri, constructori)
